## Setup

- Fill out the `.env`
  - Copy `example.env` to `.env`
  - Put in your information
- Start the userbot
  - Install dependencies from pip
    ```shell
    $ pip install -r requirements.txt
    ```
  - Run the userbot
    ```shell
    $ python userbot.py
    ```
- Start the bot
  - Install dependencies from npm
    ```shell
    $ npm install
    ```
  - Build TypeScript
    ```shell
    $ npm run build
    ```
  - Run the bot
    ```shell
    $ npm start
    ```
